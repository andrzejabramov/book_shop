## 📖 Описание проекта Bookshop

### 🔌 Использованный API

**Ресурс:** `Open Library API` (`https://openlibrary.org/search.json`)  
**Почему выбран вместо Google Books API:**  
В задании указан Google Books API, однако для фронтенд-реализации без бэкенда был выбран Open Library API. Это открытое, бесплатное API, которое:

- Не требует API-ключа и не имеет жёстких лимитов запросов.
- Работает напрямую из браузера без CORS-ограничений (не нужен прокси).
- Отдаёт структурированный JSON с обложками, авторами, ISBN и категориями.
- Полностью покрывает все функциональные требования задания.

### ✅ Реализация функциональных требований

| Требование из ТЗ                | Как реализовано                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                    |
| ------------------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **1. Шапка сайта**              | `position: sticky` + `z-index: 100` для фиксации при скролле. Логотип, навигация и кнопки-иконки. Кнопка корзины содержит динамический бейдж-счётчик, который появляется/скрывается в зависимости от `quantity` в корзине.                                                                                                                                                                                                                                                                                                                                                                                                         |
| **2. Слайдер**                  | Чистый JS. Автопрокрутка каждые 5 сек, зацикленный цикл (`index % slides.length`), навигация через точки. Справа от слайдера — статичные ссылки-блоки (пустые `href`, как по ТЗ).                                                                                                                                                                                                                                                                                                                                                                                                                                                  |
| **3. Категории и список книг**  | Динамический рендер списка категорий. Активная подсвечивается CSS-классом `.active`. При клике выполняется запрос к API с параметром `?subject={category}`. Сетка очищается, загружается первая пачка из 6 книг. Кнопка `Load more` подгружает следующие 6 без перезагрузки страницы.                                                                                                                                                                                                                                                                                                                                              |
| **4. Карточка книги**           | • **Обложка:** берётся из CDN Open Library по `cover_i`. При отсутствии — локальный плейсхолдер.<br>• **Автор/Название:** маппинг из `author_name[0]` и `title`.<br>• **Рейтинг/Отзывы:** генерируются стабильно на основе `seed` (search-эндпоинт не отдаёт точный rating). Скрываются при `0`.<br>• **Описание:** обрезается до 3 строк через CSS `line-clamp: 3` + `text-overflow: ellipsis`.<br>• **Цена:** генерируется в формате `$X.XX` (API не отдаёт коммерческие цены в поиске). Скрывается при отсутствии данных.<br>• **Кнопка:** переключает состояние `Buy now` ↔ `In the cart`, синхронизируется с модулем корзины. |
| **5. Корзина и `localStorage`** | Данные (`id`, `title`, `price`, `quantity`) сохраняются в `localStorage` под ключом `bookshop_cart`. При перезагрузке страницы состояние восстанавливается: кнопки карточек и бейдж в шапке автоматически синхронизируются с кешом.                                                                                                                                                                                                                                                                                                                                                                                                |

### 🛠️ Архитектура, сборка и оптимизация

| Аспект                  | Решение                                                                                                                                                                                                                                                                                                                                                                    |
| ----------------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **Модульная структура** | ES6 модули (`import/export`), разделение логики: `catalog-controller.js` (API + рендер), встроенный модуль `CART` (управление состоянием), BEM-нейминг для CSS.                                                                                                                                                                                                            |
| **Webpack 5**           | Разделение конфигов: `webpack.common.js` (база), `webpack.dev.js` (HMR, `style-loader`), `webpack.prod.js` (production-сборка).                                                                                                                                                                                                                                            |
| **Минификация**         | `TerserPlugin` (JS), `CssMinimizerPlugin` (CSS), `HtmlWebpackPlugin` (HTML-сжатие, автоподстановка хешей).                                                                                                                                                                                                                                                                 |
| **Оптимизации**         | • `MiniCssExtractPlugin` → выделение CSS в отдельный файл для кеширования.<br>• `CopyWebpackPlugin` → копирование статики (`img/`) в `dist/`.<br>• Code Splitting → вынос `node_modules` в `vendors.[hash].js`.<br>• `loading="lazy"` + `decoding="async"` для обложек книг.<br>• Клиентский кеш API-запросов (`Map` с TTL 5 мин) → ускорение навигации между категориями. |
